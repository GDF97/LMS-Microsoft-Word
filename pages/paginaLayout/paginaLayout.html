Página Layout

A aba "Layout" no Microsoft Word é uma das abas principais da faixa de opções e contém várias ferramentas e opções para controlar o layout, formatação e aparência do documento. Aqui estão algumas das principais funcionalidades disponíveis na aba "Layout":

Ferramentas

Margens: Você pode definir e modificar as margens do documento, alterando o espaçamento entre o texto e as bordas da página.

Tamanho: Permite definir o tamanho do papel do documento, como A4, carta, ofício, entre outros. Além disso, é possível criar tamanhos personalizados.

Orientação: Permite escolher entre a orientação retrato (vertical) e paisagem (horizontal) para o documento ou para seções específicas dele.

Colunas: É possível criar colunas no documento, dividindo o texto em duas ou mais colunas para formatar o layout.

Quebra de Página: Oferece opções para inserir quebras de página manuais, permitindo que você controle onde uma página termina e uma nova começa.

Numeração de Página: Permite adicionar números de página ao documento, personalizar o formato dos números e controlar a localização dos números de página nas páginas.

Borda da Página:Permite adicionar bordas ao redor das páginas do documento, como linhas, sombras ou estilos personalizados.

Plano de Fundo da Página:Você pode definir um plano de fundo personalizado para o documento ou para uma seção específica, como uma cor, uma imagem ou uma marca d'água.

Títulos: Facilita a aplicação de estilos de título e a criação de uma estrutura hierárquica no documento, útil para gerar sumários automáticos.

Espaçamento: Permite controlar o espaçamento entre linhas, parágrafos e margens, ajustando o espaço vertical no documento.
